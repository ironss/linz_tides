There seem to be three modules that provide access to a SQLite database
using Lua:

* luasql      http://keplerproject.org/luasql     

* lua-sqlite3 http://www.nessie.de/mroth/lua-sqlite3

* luasqlite3  http://lua.sqlite.org/index.cgi/index

* ljsqlite3   http://www.scilua.org/ljsqlite3.html


Requirements
============
* Create table
* Drop table
* Add column to table
* Remove column from table
* Start, commit, rollback transaction
* Insert, update, delete
* Prepared statements
* Parameterized statements
